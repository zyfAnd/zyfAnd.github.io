Docker的基本命令

Docker的基本命令

启动docker

 $sudo systemctl start docker
获取镜像
$docker pull [image]
$docker pull ubuntu
启动容器
docker run
查看所有的容器
$ docker ps
查看所有的镜像
$ docker images

删除所有的镜像

docker rmi $(docker images -q)

删除所有的容器

docker rm $(docker ps -aq)
